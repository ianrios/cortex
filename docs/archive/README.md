# Archive (Historical — Do Not Obey)

Pre-pivot documentation from cortex's original vision (May 2026): a Go-based
terminal-native "engineering cognition runtime" with persistent multi-layer
memory and event sourcing. On 2026-07-15 cortex was reframed as an
installable, agent-agnostic developer toolkit (see `docs/EXTRACTION_PLAN.md`).

Kept because git history is not enough for ideas this large: the memory
taxonomy, event model, and approval-gate thinking here may inform future
phases. Nothing in this directory describes the current system. Exempt from
context budgets as archived records.
