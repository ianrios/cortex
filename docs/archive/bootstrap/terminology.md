# Cortex — Terminology (v0.1)

## Core Terms

**Workspace**
- A directory-scoped cognition context

**Session**
- A bounded reasoning context with memory access

**Memory**
- Persistent structured cognition artifacts across sessions

**Intent Memory**
- The recorded "why" behind decisions and actions

**Event**
- A single atomic recorded action or decision

**Agent**
- A reasoning process operating within a session

**Tool**
- External capability exposed to agent or human

**Trace**
- Inspectable log of reasoning + execution flow
