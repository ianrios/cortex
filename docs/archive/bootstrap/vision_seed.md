# Cortex — Vision Seed (v0.1)

Cortex is a terminal-native engineering cognition runtime.

It is designed to preserve and evolve engineering understanding across time, sessions, and contexts.

## Core Vision

Software engineering today loses critical context constantly:
- why decisions were made
- how systems evolved
- what tradeoffs were considered
- what was attempted and failed

Cortex aims to preserve this missing layer of cognition.

## Principles of the System

- Continuity over stateless interaction
- Memory over repetition
- Reasoning over answers
- Inspectability over hidden intelligence
- Collaboration over autonomy
- Local-first execution and data ownership

## Long-Term Goal

Cortex should function as:
- a persistent engineering teammate
- a reasoning archive for software systems
- a living memory of project evolution
- a tool for architectural continuity across time

Not a chatbot. Not an agent demo. A cognition system.
